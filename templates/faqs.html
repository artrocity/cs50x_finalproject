Flight Search

What is the difference in a City, IATA code and ICAO code?

  • City: is the city in which the airport is located
  • IATA Code: (International Air Transport Association's Location Identifier), which is used to identify an airport. These are 3-letter codes
  • ICAO Code: (International Civil Aviation Organization) are 4-letter identifiers designed to designate airports around the world

Which API is used to search for my flights?

Currently, we are using the Tequilla by KIWI flight search API
